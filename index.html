HTML REFERENCE TABLE





Tag/Attribute Tag Format Description
Anchor <a> Creates a link
Aside <aside> Creates a sidebar that is placed next to the main content
Audio <audio> Places an audio stream within an HTML document
Body <body> Establishes the body of an HTML document
Button <button> Specifies button that can be clicked. Commonly used with forms
Column <col> Specify column headings
Dividor <div> A dividor or a kind of generic container
Emphasis <em> Emphasizes text by italicizing?
Footer <footer> The footer of a document, below the main content
Form <form> Forms for user input
Head <head> The head section. Used mainly for metadata
Header <header> Not to be confused with <head>, <header> typically contains introductory content and layout that goes above the body
HTML <html> The root level tag of an HTML document. All other HTML tags go inside the HTML tag
